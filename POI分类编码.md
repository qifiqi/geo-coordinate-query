## 使用规则

- 当前默认使用顶级编码 `090000`，所以会包含上表全部医疗相关子类。
- 若后续只要医院，可将 `医疗保健` 的编码从 `090000` 改为 `090100|090200`。
- 若只要公共服务设施，不建议加入 `070000` 生活服务；该类会包含家政、洗衣、美容、维修等大量私营商户。
- 高德可能调整或新增末级分类；查询结果中的“类型”列是 API 实际返回的权威细分类。
